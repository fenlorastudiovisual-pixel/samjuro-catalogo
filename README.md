# Samjuro Catálogo Móvil

Catálogo móvil interactivo de **Samjuro Japan S.A.S.** — diseñado como app móvil para distribuidores.

## Características

- Diseño mobile-first (max-width 430px)
- Bottom sheets, swipe gestures, animaciones
- Panel de pedido + favoritos + perfil
- Integración con WhatsApp para cotización
- Preparado para integración con Wompi

## Deploy

Auto-deploy a Cloudflare Pages al hacer push a `main`.
